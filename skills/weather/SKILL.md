# 天气查询技能

## 功能

查询全球任何城市的天气信息，包括：

- 当前温度
- 体感温度
- 湿度
- 天气状况
- 风速

## 使用示例

- "北京天气怎么样？"
- "查询纽约的天气"
- "上海今天冷吗？"

## 数据来源

使用免费的 Open-Meteo API，无需 API Key。
