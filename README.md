### 快速开始

```bash
# 克隆项目
git clone https://github.com/Memesclaw/memeclaw.git
cd memeclaw

# 安装依赖
npm install

# 编译项目
npm run build

# 自动配置
npm run onboard

# 启动服务
npm start
```

启动后访问：
- **Web 界面**: http://127.0.0.1:18789
- **Telegram Bot**: `@your_bot_username`

---

### 配置说明

```bash
# 自动配置向导
npm run onboard

# 或手动创建配置文件
mkdir ~/.memeclaw
```

配置文件示例 (`~/.memeclaw/config.json`):
```json
{
  "api": {
    "mode": "relay",
    "endpoint": "https://your-api.com",
    "key": "sk-xxx",
    "model": "claude-opus-4-6"
  },
  "telegram": {
    "enabled": true,
    "botToken": "your-bot-token"
  }
}
```

### 常见问题

<details>
<summary><b>如何获取 Telegram Bot Token？</b></summary>

1. 在 Telegram 中搜索 `@BotFather`
2. 发送 `/newbot` 命令
3. 按提示设置 Bot 名称
4. 获取 Token 并填入配置文件
</details>

<details>
<summary><b>如何添加自定义技能？</b></summary>

1. 在 `skills/` 目录下创建新文件夹
2. 编写 `SKILL.md` 定义技能
3. 重启服务即可加载
4. 详见 [技能开发指南](docs/SKILL_GUIDE.md)
</details>
